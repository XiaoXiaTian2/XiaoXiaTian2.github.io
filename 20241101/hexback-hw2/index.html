# 小節作業二 - 主鍵、外來鍵、inner join

# 1. 哪個欄位可以做為外鍵

# Ans：將班級做成外鍵

  • 建立班級 (classses) 表
  • 建立學生 (students) 表並使用外鍵參考 classes 的 id
  • 新增資料
  • 透過 INNER JOIN 查詢

# 2. 接續第一題，但多增加 班級老師

# 同樣拆出 班級，班級老師 也可同樣拆成一張表

  • 建立班級老師 (teachers) 表
  • 建立班級 (classes) 表
  • 建立學生 (students) 表，並使用外鍵參考 classes、teachers 的 id
  • 建立三張表的資料
  • 利用 INNER JOIN 查詢結果

# 3. 家庭資料表，減少重複

# 將 父母相關資料 皆拆出

  • 建立父母 (parents) 表
  • 建立孩子 (children) 表，並使用外鍵參考 parents 的 id
  • 新增 parents、children 資料
  • 利用 INNER JOIN 查詢結果
